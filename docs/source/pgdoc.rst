``rawseeds_camera_mean_generalized``
------------------------------------------------------------
Computes and displays the mean of an arbitrary block.


.. rst-class:: procgraph:config

Configuration
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

- ``logdir``: Rawseeds log directory

- ``prefix``: Prefix for this operation (writes in $logdir/prefix)
